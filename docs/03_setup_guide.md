# 环境配置

## 建议
1. 使用独立虚拟环境
2. 安装基础分析库
3. 先验证 Akshare，再验证 Tushare
4. 保存结果到 docs/

## 存储约束
- 当前本地数据库与存储容量：不足 512G
- 后期计划扩展到 1T 数据
- 第一阶段优先保留：信号结果、持仓记录、近期行情缓存、自选池、指标结果
- 原始历史数据建议分层归档到 NAS / 云盘 / 外置存储
- 避免一开始将全量历史行情长期堆积在本地库

## 数据源频次约束
- Tushare 的 `daily` 可作为主日线源
- `stock_basic` 属于低频接口，建议缓存优先、按需刷新，不要放进高频流水线
- `stock_basic` 刷新采用缓存 TTL 策略，默认 24 小时
- AKShare 的部分股票行情接口目前不稳定，指数接口可作为补充
